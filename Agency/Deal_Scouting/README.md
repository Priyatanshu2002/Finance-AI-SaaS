# Module D: Deal Scouting Engine
> **Status:** Phase 5 (Planned)
> **Output:** Qualified Deal Flow List (Excel)

## Workflow
1.  **Targeting:** Define Niche (e.g., "Dental Practices >$1M Revenue in FL").
2.  **Scraping:** Firecrawl crawls logic to find targets.
3.  **Enrichment:** Opus qualifies leads (Owner Age, Reviews, Red Flags).
4.  **Outreach:** Generates customized cold emails.

## File Structure
*   `scout.py`: Main scraper script.
*   `/Leads`: Database of scraped targets.
